# 🌐 Multilingual SEO Architecture

The project uses a professional multilingual URL structure:

```bash
/en/
/ru/
/uz/
```

Example:

```bash
/en/tools/image-to-pdf
/ru/tools/image-to-pdf
/uz/tools/image-to-pdf
```

Benefits:

- Better Google indexing
- hreflang support
- localized SEO
- scalable architecture
- international accessibility

---

# 🌎 Automatic Language Detection

The platform automatically detects the user's browser language.

### Example

| Browser Language | Redirect |
|---|---|
| Russian | `/ru/` |
| Uzbek | `/uz/` |
| Other | `/en/` |

Implemented using:

```javascript
navigator.language
```

Additional features:
- localStorage persistence
- manual language switching
- fallback language system
