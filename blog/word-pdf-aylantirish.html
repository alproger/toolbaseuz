Word dan PDF ga Aylantirish: DOCX, DOC ni PDF qilish

Formatlash saqlanadi, shrift o’zgarmaydi. Brauzerda bepul — serverga yuborilmaydi.

PDF — hujjatni yuborish va chop etishning universal standarti. Word (DOCX) faylini hamma qurilmada bir xil ko’rsatish, tahrirlashdan himoya qilish yoki email ga biriktirish uchun PDF ga aylantirish kerak bo’ladi.

Word hujjatini hoziroq PDF ga aylantiring

DOCX, DOC → PDF. Formatlash, jadvallar, ro’yxatlar saqlanadi. Bepul, brauzerda.

toolbase.uz Word → PDF qanday ishlaydi?

toolbase.uz vositasi mammoth.js kutubxonasidan foydalanadi. Bu kutubxona DOCX faylini to’g’ridan brauzerda o’qib, HTML ga aylantiradi. Keyin brauzerning print API si orqali PDF yaratiladi.

  • Sarlavhalar (H1–H6) — saqlanadi
  • Jadvallar — to’liq ko’chiriladi
  • Ro’yxatlar (ol, ul) — saqlanadi
  • Qalin (bold), kursiv (italic) — saqlanadi
  • Rasmlar — qo’llab-quvvatlanadi

Cheklov: Murakkab Word formatlash (ustunlar, watermark, maxsus shriftlar) to’liq ko’chirilmasligi mumkin. Oddiy hujjatlar (rezume, ariza, shartnoma) uchun natija a’lo.

Qanday foydalanish?

  1. DOCX yoki DOC faylni yuklang — drop zonaga tashlang yoki «Fayl tanlash» tugmasi.
  2. Preview ko’ring — hujjat brauzerda ko’rsatiladi.
  3. «PDF yuklab olish» tugmasini bosing — PDF fayl qurilmangizga yuklanadi.

Maslahat: Murakkab formatlash bo’lsa, avval Word da «Fayl → Eksport → PDF/XPS» orqali saqlang. Bu usul eng aniq natija beradi.

Boshqa Word → PDF usullari

Microsoft Word (to’g’ridan)

Word dasturi ichida: Fayl → Saqlash nusxasi → PDF yoki Fayl → Eksport → PDF/XPS yaratish. Eng aniq natija bu usulda.

Google Docs

DOCX ni Google Docs ga yuklang → Fayl → Yuklab olish → PDF hujjati. Bepul, lekin internet kerak va fayl Google serverlariga yuklanadi.

LibreOffice (offline)

Bepul ochiq manba ofis paketi. Fayl → Eksport PDF sifatida. Microsoft Office bo’lmaganlar uchun eng yaxshi offline yechim.

Ko’p so’raladigan savollar

Fayl serverga yuboriladi? Yo’q. toolbase.uz vositasi to’liq brauzerda ishlaydi — mammoth.js DOCX ni lokal o’qiydi.

Qanday fayl hajmiga chidaydi? Brauzer xotirasiga bog’liq. Odatda 50–100 MB gacha muammosiz ishlaydi.

DOC formatini ham qabul qiladi? DOCX (Office 2007+) asosiy format. Eski DOC fayllar uchun avval DOCX ga saqlash tavsiya qilinadi.
