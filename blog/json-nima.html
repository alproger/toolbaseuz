JSON nima va u nima uchun ishlatiladi? — Boshlovchilar uchun to'liq qo'llanma

JSON — internetdagi ma'lumot almashishning asosi. Saytlar, ilovalar va API'lar bir-biri bilan JSON orqali gaplashadi. Uni tushunish har bir dasturchi va texnik mutaxassis uchun muhim.

1. JSON — bu nima?

JSON (JavaScript Object Notation — JavaScript Obyekt Yozuvi) — ma'lumotlarni matn ko'rinishida ifodalash va almashish uchun standart format. 2001-yilda Douglas Crockford tomonidan yaratilgan va hozir internet ma'lumot almashishining asosiy tiliga aylangan.

JSON ning afzalliklari:

  • Inson o'qiy oladi — oddiy matn, har qanday matn muharririda ko'rish mumkin
  • Engil — XML ga qaraganda ancha ixcham
  • Universal — barcha zamonaviy dasturlash tillari JSON ni qo'llab-quvvatlaydi
  • Tez — parsing (tahlil qilish) va generatsiya tez ishlaydi

2. JSON qanday ko'rinadi?

Oddiy JSON misoli — bir o'quvchi haqida ma'lumot:

{
  "ism": "Sardor",
  "familiya": "Yusupov",
  "yosh": 22,
  "talaba": true,
  "manzil": {
    "shahar": "Toshkent",
    "tuman": "Yunusobod"
  },
  "fanlar": ["matematika", "fizika", "informatika"],
  "stipendiya": null
}

4. JSON qayerda ishlatiladi?

4.1. REST API'lar

Zamonaviy veb va mobil ilovalar backend server bilan JSON orqali muloqot qiladi. Masalan, ob-havo ilovasida: 

{
  "shahar": "Toshkent",
  "harorat": 32,
  "birlik": "Celsius",
  "holat": "quyoshli"
}

4.2. Konfiguratsiya fayllari

Ko'pgina dasturlar va vositalar JSON formatida konfiguratsiya fayllaridan foydalanadi: package.json (Node.js), tsconfig.json (TypeScript), settings.json (VS Code) va boshqalar.

4.3. Ma'lumotlar bazasi

PostgreSQL, MySQL va MongoDB kabi ma'lumotlar bazalari JSON ustunlarini qo'llab-quvvatlaydi — tuzilmasi o'zgaruvchan ma'lumotlarni saqlash uchun qulay.

4.4. Fayl formatlari

GeoJSON (xarita ma'lumotlari), JSON-LD (semantik veb), HAR (HTTP arxiv) — barchasi JSON asosida qurilgan maxsus formatlar.

5. JSON va XML — taqqoslash

JSON dan avval XML keng qo'llanilgan. Ikkala formatda bir xil ma'lumot:

/* XML — 156 belgi */
<foydalanuvchi>
  <ism>Ali</ism>
  <yosh>25</yosh>
</foydalanuvchi>

/* JSON — 32 belgi */
{"ism": "Ali", "yosh": 25}

JSON ancha ixcham, o'qilishi oson va tezroq tahlil qilinadi.

6. JSON da ishlash — amaliy misollar

JavaScript da

// JSON stringni obyektga aylantirish
const jsonMatn = '{"ism":"Ali","yosh":25}';
const obyekt = JSON.parse(jsonMatn);
console.log(obyekt.ism); // "Ali"

// Obyektni JSON stringga aylantirish
const json = JSON.stringify(obyekt, null, 2);

Python da

import json

# JSON stringni dict ga aylantirish
matn = '{"ism": "Ali", "yosh": 25}'
lug_at = json.loads(matn)
print(lug_at["ism"])  # Ali

# Dict ni JSON ga aylantirish
json_matn = json.dumps(lug_at, ensure_ascii=False, indent=2)

💡 ensure_ascii=False: Python da JSON yozayotganda ensure_ascii=False parametrini qo'shing — aks holda o'zbek harflari (oʻ, gʻ) \uXXXX kodiga aylantiriladi.

7. JSON ni tekshirish va formatlash

JSON yozishda kichik xato ham butun hujjatni noto'g'ri qiladi. Qo'shtirnoq, vergul, qavs — har biri muhim. Onlayn JSON formatlash vositasi xatolarni aniqlab, qaysi qator va ustunda ekanini ko'rsatadi.
