Qanday foydalanish?

  1. Rasmlarni yuklang — drag & drop yoki tugma bilan.
  2. Ro’yxatdan rasmni bosib tanlang — o’ng panelda preview yangilanadi.
  3. Sifat slayderi bilan 82% da boshlang — before/after preview bilan solishtiring.
  4. Barcha Rasmlarni Siqish tugmasini bosib yuklab oling.
Qaysi sifat darajasini tanlash kerak?

Veb va ijtimoiy tarmoq uchun 80–85% optimal. Ko'z bilan farq qilish qiyin, lekin hajm 2 marta kamayadi. Bosmaxona uchun 95–100%.

WebP formatini tanlash kerakmi?

WebP JPG dan 25–35% kichikroq. Barcha zamonaviy brauzerlar qo'llab-quvvatlaydi. Agar sayt yoki dasturingiz WebP qabul qilsa — tanlang.

PNG siqiladi?

PNG lossless format — sifat o'zgarmaydi. Lekin WebP ga aylantirish orqali hajmni 50–70% kamaytirish mumkin.

Rasmlar serverga yuboriladi?

Yo'q. Canvas API to'liq brauzeringizda ishlaydi. Hech qanday ma'lumot yuborilmaydi.
