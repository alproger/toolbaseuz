Algoritmlar

  • MD5 — 32 hex. Kriptografik jihatdan zaif, lekin checksum va cache key uchun keng ishlatiladi.
  • SHA-1 — 40 hex. Git commit ID larda ishlatiladi. MD5 dan mustahkam.
  • SHA-256 — 64 hex. Tavsiya etilgan. Blockchain, TLS, parol hashlash standarti.
  • SHA-512 — 128 hex. Eng kuchli, 64-bit protsessorlarda tez ishlaydi.
MD5 xavfsizmi?

Kriptografik maqsadlar uchun xavfsiz emas. Lekin fayl checksum uchun yetarli.

Qaysi algoritmni tanlash kerak?

Umumiy maqsad uchun SHA-256 tavsiya etiladi. Yuqori xavfsizlik uchun SHA-512.
