Parol turlari

Foydalanuvchi paroli — hujjatni ochish uchun talab qilinadi. Bu parolni bilmagan kishi PDF ni ko’ra olmaydi.

Egasi paroli — to’liq huquqlar uchun. Foydalanuvchi paroli bilan ochilgan hujjatda chop etish, tahrirlash cheklangan bo’lishi mumkin. Egasi paroli bilan barcha cheklovlar olib tashlanadi.

Parolni unutsam nima bo'ladi?

Parolni qayta tiklash imkoniyati yo'q. Iltimos, parolingizni xavfsiz joyda saqlang. toolbase.uz fayl yoki parolni saqlamaydi.

Qaysi shifrlash ishlatiladi?

@cantoo/pdf-lib kutubxonasi PDF standartiga mos AES-128 shifrlashni qo'llaydi. Zamonaviy PDF ko'ruvchilar (Adobe Acrobat, Chrome, Firefox) bu formatni qo'llab-quvvatlaydi.

Himoyasiz PDFdan parolni olib tashlash mumkinmi?

Ha. Agar fayl himoyasiz bo'lsa, parol bo'sh qoldiriladi va fayl aynan shu holida saqlanadi.

Ruxsatlar qanday ishlaydi?

Foydalanuvchi parolida ochilganda belgilangan ruxsatlar qo'llaniladi. Egasi paroli bilan ochilganda barcha ruxsatlar avtomatik beriladi. Ba'zi eski PDF ko'ruvchilar ruxsatlarni to'liq qo'llamasligi mumkin.

Fayl brauzerdan tashqariga chiqadi?

Yo'q. Hamma jarayon JavaScript va Web Crypto API orqali foydalanuvchi qurilmasida bajariladi. Fayl hech qachon serverga yuborilmaydi.
