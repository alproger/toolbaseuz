Объединение нескольких PDF файлов в один документ — самая популярная операция с PDF.aridan biri.

Если PDF показывает «защищённый» — снимите пароль перед объединением.ом.lishi mumkin. Bunday файлыni birlashtirish uchun парольni olib перетащите или OCR vositasidan foydalaning.

Qanday foydalanish?

  1. Перетащите PDF файлы в поле выше или нажмите «Выбрать файлы».
  2. Файлы ro'yxatda ko'rinadi. Sudrab istalgan tartibga soling.
  3. "Объединение PDF" нажмите кнопку.
  4. Результат готов bo'lgach "Скачать" tugmasi paydo bo'ladi.

Ko'p so'raladigan savollar

PDF файлыim xavfsizmi?

Да, to'liq xavfsiz. Все qayta ishlash faqat в вашем браузере pdf-lib kutubxonasi orqali amalga oshiriladi.

Сколько PDF можно объединить?

Bir vaqtda 20 макс PDF файл birlashtiriladi. Har bir файл 100 MB dan oshmоригиналigi kerak.

Парольli PDF bilan ishlaydi?

PDF с паролем показывает количество страниц, но для объединения нужно сначала снять пароль.arolni olib перетащите или парольni введите.

Можно ли изменить порядок страниц?

Ha! Файл kartalarini sudrab (drag & drop) istalgan tartibga qo'yishingiz mumkin.
