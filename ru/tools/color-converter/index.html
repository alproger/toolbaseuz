Rang formatlari haqida

HEX — veb dizaynda eng keng tarqalgan. #rrggbb formatida, r/g/b — qizil/yashil/ko’k 00–ff orasida.

RGB — qizil, yashil, ko’k. Har biri 0–255. Ekranlar va CSS uchun standart.

HSL — Hue (rang turi), Saturation (to’yinganlik), Lightness (yorqinlik). Inson idrokiga yaqin, CSS da juda qulay.

CMYK — Cyan, Magenta, Yellow, Key (qora). Bosmaxona standart formati.

HSV — Hue, Saturation, Value. Photoshop va boshqa grafik dasturlarda ishlatiladigan format.

CSS gradient nima?

CSS gradient — ikki yoki undan ko’p rangning silliq o’tishi. background: linear-gradient(135deg, #0891b2, #7c3aed) kabi CSS xususiyati orqali belgilanadi. Rasm fayli kerak emas.

HEX va RGB qaysi birini ishlatish kerak?

Veb uchun ikkisi ham to’g’ri. HSL CSS da rang sistеmasini boshqarish uchun qulay — yorqinlikni osonroq o’zgartirish mumkin. Bosmaxona uchun CMYK kerak.

Nomlangan ranglar qancha?

CSS standartida 140+ nomlangan rang bor (red, blue, tomato va boshqalar). Vositada 36 ta eng ko’p ishlatiladigan rang o’zbek nomda keltirilgan.
