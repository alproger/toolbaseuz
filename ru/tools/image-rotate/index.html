Частые вопросы

Можно ли повернуть несколько файлов?

Инструмент работает с одним файлом за раз.

Сохраняется ли итог?

Да, итоговый файл сохраняется после экспорта.

Есть ли мобильная версия?

Да, инструмент подходит для смартфонов.
