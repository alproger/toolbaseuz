Qanday foydalanish?

Визуальный выбор usuli:

  1. Загрузите PDF файл — страницы отобразятся в виде миниатюр.
  2. Нажмите страницы для удаления (красный = удаляется).
  3. "Удалить выбранные страницы" нажмите кнопку.
  4. Результатni yuklab oling.

Raqam bilan tanlash usuli:

  1. Загрузите PDF файл.
  2. "Raqam bilan tanlash" tabini нажмите.
  3. "Dan" va "Gacha" maydonlariga страница raqamlarini введите.
  4. "Выбрать" tugmasini нажмите — страницы avtomatik tanlanadi.
  5. "Удалить выбранные страницы" нажмите кнопку.

Ko'p so'raladigan savollar

Все страницыni o'chirib bo'ladimi?

Нет. Kamida 1 ta страница qolishi kerak. Все страницыni o'chirishga urinilsa, xato xabari ko'rinadi.

Nechta страница o'chirish mumkin?

Kamida 1 ta страница qolguncha istalgancha страница o'chirish mumkin.

Парольli PDF bilan ishlaydi?

Нет. Пароль bilan himoyalangan PDF файлы ochilmaydi. Avval парольni olib перетащите.

Страницы preview ko'rsatiladi?

Да! PDF.js рендерит миниатюру каждой страницы прямо в браузере.
