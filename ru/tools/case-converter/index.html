Formatlar

UPPER CASE — barcha harflar katta. lower case — barcha kichik. Title Case — har so‘z katta harf. Sentence case — faqat birinchi harf katta. camelCase — dasturlash uchun. PascalCase — sinflar uchun. snake_case — Python, DB. kebab-case — CSS, URL.

camelCase va PascalCase farqi?

camelCase: salomDunyo — birinchi so‘z kichik. PascalCase: SalomDunyo — barcha so‘zlar katta harf bilan. JavaScript o‘zgaruvchilari camelCase, React komponentlari PascalCase.
