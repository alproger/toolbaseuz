PDF файлыni пароль bilan himoyalash muhim hujjatlarni ruxsatsiz kirishdan saqlaydi. Bu vosita AES shifrlash orqali работает в вашем браузере — файл не отправляется на сервер.

Пароль turlari

Foydalanuvchi парольi — hujjatni ochish uchun talab qilinadi. Bu парольni bilmagan kishi PDF ni ko’ra olmaydi.

Egasi парольi — to’liq huquqlar uchun. Foydalanuvchi парольi bilan ochilgan hujjatda chop etish, tahrirlash cheklangan bo’lishi mumkin. Egasi парольi bilan все cheklovlar olib tashlanadi.

Парольni unutsam nima bo'ladi?

Парольni qayta tiklash imkoniyati yo'q. Iltimos, парольingizni xavfsiz joyda saqlang. toolbase.uz файл или парольni saqlamaydi.

Qaysi shifrlash ishlatiladi?

@cantoo/pdf-lib kutubxonasi PDF standartiga mos AES-128 shifrlashni qo'llaydi. Zamonaviy PDF ko'ruvchilar (Adobe Acrobat, Chrome, Firefox) bu forтекстi qo'llab-quvvatlaydi.

Himoyasiz PDFdan парольni olib tashlash mumkinmi?

Да. Agar файл himoyasiz bo'lsa, пароль bo'sh qoldiriladi va файл aynan shu holida saqlanadi.

Ruxsatlar qanday ishlaydi?

Foydalanuvchi парольida ochilganda belgilangan ruxsatlar qo'llaniladi. Egasi парольi bilan ochilganda все ruxsatlar avtomatik beriladi. Ba'zi eski PDF ko'ruvchilar ruxsatlarni to'liq qo'llamоригиналigi mumkin.

Файл покидает браузер?

Нет. Hamma jarayon JavaScript va Web Crypto API orqali foydalanuvchi qurilmasida bajariladi. Файл hech qachon на сервер не отправляется.
