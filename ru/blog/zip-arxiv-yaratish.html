Создание ZIP-архива: упаковка файлов и папок

Собрать много файлов в одном месте для отправки, сделать резервную копию или уменьшить размер — правильные способы создания ZIP-архива.

ZIP-архив — универсальный формат, упаковывающий несколько файлов в один со сжатием. Стандарт с 1989 года, поддерживается всеми операционными системами и устройствами. Самый распространённый способ прикрепить много файлов к письму, заархивировать проект или отправить большую папку.

DEFLATE и STORE: что выбрать?

Характеристика DEFLATE STORE
Сжатие ✓ Bor ✗ Yo’q
Уменьшение размера 30–70% (текст, kod) 0%
Скорость Медленнее Tezroq
Для JPG/MP4/ZIP Мало пользы Afzal
Для TXT/PDF/DOCX Afzal Мало пользы

Правило: текст, код, документы — DEFLATE. Уже сжатые файлы (изображения, видео, архивы) — STORE. Если смешано — DEFLATE (вреда не будет).

Сохранение структуры папок

Когда включён переключатель «Структура папок», внутри ZIP сохраняется вид папка/подпапка/файл.txt. Это важно — при открытии ZIP файлы будут лежать в своих папках.

Если переключатель выключен — все файлы располагаются плоско в корне ZIP. Иногда это удобно — например, когда нужно собрать в одном месте файлы из нескольких папок.

Способы добавления папки

1. Кнопка «Выбрать папку» (все браузеры)

С атрибутом webkitdirectory диалог выбора файлов переключается на выбор папки. Загружается полная структура с вложенными папками и файлами. Chrome, Firefox, Edge, Safari — всё работает.

2. Drag & drop (Chrome, Edge)

Перетаскивание папки прямо в зону drop — этот способ работает через API webkitGetAsEntry() и рекурсивно читает подпапки. В Firefox перетаскивание папки не работает — используйте кнопку «Выбрать папку».

Для пользователей Firefox: добавление папки перетаскиванием в Firefox не поддерживается. Всегда используйте кнопку «Выбрать папку» — она работает во всех браузерах.

Коэффициент сжатия: чего ожидать?

  • Текстовые файлы (TXT, CSV, JSON, XML): сжимаются на 60–90% — наилучший результат
  • Документы (DOCX, XLSX, PPTX): 10–40% — уже сжатый формат
  • PDF: 5–20% — kontentga bog’liq
  • Kod fayllari (JS, PY, HTML, CSS): 50–80%
  • Изображения (JPG, PNG, WebP): 0–5% — уже сжаты
  • Видео (MP4, MKV): 0–2% — пользы нет

Важно: для изображений и видео сжатие ZIP почти не даёт пользы — они уже сжаты в своём формате. Для таких файлов режим STORE работает быстрее, а результат тот же.

Как пользоваться?

  1. Перетащите файлы в зону drop или нажмите кнопку «Добавить файл».
  2. Чтобы добавить папку, нажмите кнопку «Выбрать папку».
  3. Просмотрите содержимое в дереве файлов — удалите ненужные файлы.
  4. Введите имя ZIP, выберите тип сжатия.
  5. Нажмите «Создать ZIP» и скачайте.
