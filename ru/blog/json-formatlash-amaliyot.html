Почему форматировать JSON

Отформатированный JSON легче читать и отлаживать, а в production используется минификация.

Типичные ошибки

Следите за отсутствующими запятыми, одинарными кавычками и комментариями, которые не поддерживаются.

Работа с инструментом

Частые вопросы

Что такое minify?

Minify уменьшает размер JSON за счет удаления пробелов.

Можно ли валидировать неверный JSON?

Да, инструмент покажет ошибки.
