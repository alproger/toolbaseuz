Цветовые форматы: HEX, RGB, HSL, CMYK и CSS-градиент

Какой формат когда использовать, в чём их различия и как правильно создавать CSS-градиент.

Цветовые форматы — подробно

Веб

HEX

#0891b2
#ff6b35
#fff

Шестнадцатеричная система. Состоит из 6 или 3 символов. Стандартный формат веба и графических программ.

CSS

RGB

rgb(8, 145, 178)
rgb(255, 107, 53)

Красный, зелёный, синий. Диапазон 0–255. Физическая модель цвета для экранов.

CSS3

HSL

hsl(192, 91%, 36%)
hsl(16, 100%, 60%)

Тон, насыщенность, яркость. Ближе всего к человеческому восприятию. Легко управлять в CSS.

Печать

CMYK

cmyk(96%, 19%, 0%, 30%)
cmyk(0%, 58%, 79%, 0%)

Стандарт для принтеров и типографий. Модель смешения красок.

Когда какой формат использовать?

Цель Формат Причина
Веб CSS HEX или HSL HEX — кратко, HSL — удобно управлять
CSS-анимация RGB или HSL Плавный переход между значениями
Типография CMYK Модель красок принтера
Photoshop/GIMP HSV или RGB Стандарты графических программ
Figma/Sketch HEX или RGB Экспорт из дизайн-программ
Цветовая система HSL Легко создавать светлее/темнее

HSL — почему он самый удобный?

HSL (Hue, Saturation, Lightness) — самый интуитивный формат для выбора и управления цветом. Особенно полезен при создании цветовой системы:

  • hsl(192, 91%, 36%) — основной цвет
  • hsl(192, 91%, 50%) — светлее (L увеличен)
  • hsl(192, 91%, 20%) — темнее (L уменьшен)
  • hsl(192, 30%, 36%) — приглушённее (S уменьшен)

Практическая польза: при использовании HSL с CSS-переменными всей цветовой системой можно управлять, меняя только значение L. Tailwind CSS тоже использует HSL.

CSS-градиент — полное руководство

CSS-градиент — возможность создать фон без файла изображения. Существует три вида:

1. Linear gradient (линейный)

Цвета переходят вдоль прямой линии:

background: linear-gradient(135deg, #0891b2 0%, #7c3aed 100%);

2. Radial gradient (радиальный)

Цвета расходятся от центра наружу:

background: radial-gradient(circle, #f97316 0%, #ec4899 100%);

3. Conic gradient (конический)

Цвета вращаются вокруг центра — цветовое колесо:

background: conic-gradient(from 0deg, #0891b2, #7c3aed, #ec4899, #0891b2);

Формулы конвертации цвета

Из HEX в RGB: #0891b2 → 08=8, 91=145, b2=178 → rgb(8, 145, 178)

Из RGB в HEX: перевод r, g, b в шестнадцатеричную систему → #rrggbb

Из RGB в CMYK: k = 1 − max(r,g,b)/255; c = (1−r/255−k)/(1−k) ...

На практике: эти вычисления не нужно делать вручную. Конвертер цвета toolbase.uz автоматически переводит между всеми форматами — просто введите значение.

Частые вопросы

В HEX важен регистр букв? Нет. #0891B2 и #0891b2 — одно и то же.

Разница между #fff и #ffffff? Один и тот же цвет. Короткая 3-символьная запись работает только для парных символов (ff, aa, 00).

Что такое rgba()? RGB + Alpha (прозрачность). rgba(8, 145, 178, 0.5) — цвет с 50% прозрачностью.

Совместимость CSS-градиента с браузерами? Все современные браузеры поддерживают. Для старого IE нужен был префикс -webkit- — генератор toolbase.uz добавляет его автоматически.
