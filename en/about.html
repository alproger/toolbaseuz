About Us

toolbase.uz — Free online tools platform created for Uzbek users to work with text and documents.

About the Project

toolbase.uz is a free platform created to help Uzbek users work with text, documents, and files. All tools run directly in your browser — no installation or registration required.

Our Mission

We believe every Uzbek user should have access to fast, quality, and free online tools. Latin-Cyrillic converter, number-to-words converter, word counter, and many other practical tools — all in one place, fast and reliable.

Available Tools

  • Latin ⇄ Cyrillic — Quickly convert Uzbek text between two scripts
  • Number → Words — Convert numbers to Uzbek words (for financial documents)
  • Word Counter — Count words, characters, sentences and reading time
  • Case Converter — UPPER, lower, Title, camelCase, snake_case and more
  • Password Generator — Cryptographically secure strong password generator
  • Image Tools — Compress, resize, crop, rotate and convert images
  • PDF Tools — Merge, split, protect and edit PDF files
  • And more... — Color converter, QR generator, unit converter, and others

Technology

All tools are built with pure HTML, CSS and JavaScript — no server-side processing. Your data is never sent to the server and remains completely private.
