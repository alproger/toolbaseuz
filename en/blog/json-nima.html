What is JSON? — Complete Beginner's Guide

JSON is the foundation of data exchange on the web. Websites, apps, and APIs communicate using JSON, so understanding it is essential for developers.

1. What is JSON?

JSON (JavaScript Object Notation) is a text format used to represent data in a simple, readable way. It became popular in the early 2000s and is now the standard for APIs and configuration files.

JSON is lightweight, human-readable, and supported by all major programming languages.

2. JSON structure

JSON uses two main building blocks:

  • Objects — collections of key/value pairs enclosed in curly braces.
  • Arrays — ordered lists of values enclosed in square brackets.

Here is a sample JSON object describing a student:

{
  "name": "John",
  "surname": "Smith",
  "age": 22,
  "student": true,
  "address": {
    "city": "New York",
    "state": "NY"
  },
  "subjects": ["math", "physics", "computer science"],
  "scholarship": null
}

4. Where JSON is used

4.1. REST APIs

Web and mobile applications exchange data with servers using JSON. For example, a weather API response might look like this:

{
  "city": "San Francisco",
  "temperature": 68,
  "unit": "Fahrenheit",
  "condition": "sunny"
}

4.2. Configuration files

Tools and frameworks often store settings in JSON files such as package.json, tsconfig.json, and settings.json.

4.3. Databases

Many databases support JSON data types, letting you store flexible records without a fixed schema.

4.4. Specialized formats

Formats such as GeoJSON, JSON-LD, and HAR are built on JSON to describe maps, structured data, and network logs.

5. JSON vs XML

JSON is generally more compact and easier to read than XML. Example:

/* XML — 156 characters */
<user>
  <name>Alice</name>
  <age>25</age>
</user>

/* JSON — 32 characters */
{"name": "Alice", "age": 25}

JSON is shorter, easier to parse, and preferred for modern web applications.

6. Working with JSON

JavaScript

// Convert JSON string to object
const jsonString = '{"name":"Bob","age":30}';
const obj = JSON.parse(jsonString);
console.log(obj.name); // Bob

// Convert object to JSON string
const json = JSON.stringify(obj, null, 2);

Python

import json

# Convert JSON string to dict
text = '{"name": "Bob", "age": 30}'
data = json.loads(text)
print(data["name"])  # Bob

# Convert dict to JSON
json_text = json.dumps(data, indent=2)

Tip: JSON is strict — a missing comma or quote will break the entire document.
