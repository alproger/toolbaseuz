JSON Basics

JSON stands for JavaScript Object Notation. It uses key-value pairs, arrays, and simple data types for easy data exchange.

Valid JSON Rules

Use double quotes around keys and strings, avoid trailing commas, and only include supported types such as numbers, strings, booleans, null, arrays and objects.

Why JSON is Popular

JSON is human-readable, compact, and supported by nearly every programming language. It is ideal for APIs, config files, and data storage.

Frequently asked questions

Can JSON contain comments?

No, standard JSON does not support comments.

Is JSON the same as JavaScript object?

JSON is a text format, while JavaScript objects are runtime values.
