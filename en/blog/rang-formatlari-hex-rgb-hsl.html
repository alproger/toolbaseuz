HSL — nima uchun eng qulay?

HSL (Hue, Saturation, Lightness) — rang tanlash va boshqarishda eng intuitiv format. Rang tizimini yaratishda ayniqsa foydali:

  • hsl(192, 91%, 36%) — asosiy rang
  • hsl(192, 91%, 50%) — ochroq variant (L oshirildi)
  • hsl(192, 91%, 20%) — to’qroq variant (L kamaytirildi)
  • hsl(192, 30%, 36%) — o’chiqroq variant (S kamaytirildi)

Amaliy foyda: CSS o’zgaruvchilari bilan HSL ishlatganda, faqat L qiymatini o’zgartirish orqali butun rang tizimini boshqarish mumkin. Tailwind CSS ham HSL dan foydalanadi.

CSS Gradient — to’liq qo’llanma

CSS gradient — image faylisiz fon yaratish imkoniyati. Uch turi mavjud:

1. Linear gradient (chiziqli)

Ranglar to’g’ri chiziq bo’ylab o’tadi:

background: linear-gradient(135deg, #0891b2 0%, #7c3aed 100%);

2. Radial gradient (markazdan)

Ranglar markazdan tashqariga yoyiladi:

background: radial-gradient(circle, #f97316 0%, #ec4899 100%);

3. Conic gradient (konussimon)

Ranglar markaz atrofida aylanadi — rang g’ildiragi:

background: conic-gradient(from 0deg, #0891b2, #7c3aed, #ec4899, #0891b2);

Rang konvertatsiya formulalar

HEX dan RGB: #0891b2 → 08=8, 91=145, b2=178 → rgb(8, 145, 178)

RGB dan HEX: r, g, b ni 16-lik tizimga o’girish → #rrggbb

RGB dan CMYK: k = 1 − max(r,g,b)/255; c = (1−r/255−k)/(1−k) ...

Ko’p uchraydigan savollar

HEX katta-kichik harflar farq qiladimi? Yo’q. #0891B2 va #0891b2 bir xil.

#fff va #ffffff farqi? Bir xil rang. 3 belgilik qisqa yozuv faqat juft harflar (ff, aa, 00) uchun ishlaydi.

rgba() nima? RGB + Alpha (shaffoflik). rgba(8, 145, 178, 0.5) — 50% shaffof rang.

CSS gradient browser mosligi? Barcha zamonaviy brauzerlar qo’llab-quvvatlaydi. Eski IE uchun -webkit- prefiks kerak edi — toolbase.uz generator avtomatik qo’shib beradi.
