Rasmni Aylantirish haqida

Canvas API orqali to’liq brauzerda ishlaydi — rasm serverga yuborilmaydi. Oddiy 90° aylantirish tugmalaridan tortib erkin −180°dan +180° gacha burchak, gorizontal/vertikal flip va fon rangi tanlash imkoniyati.

Qanday foydalanish?

  1. Rasmlarni yuklang — drag&drop yoki tugma orqali.
  2. Ro’yxatdan rasmni bosib tanlang — o’ng tomonda preview yangilanadi.
  3. Burchak tanlang — tez tugmalar yoki slider bilan.
  4. Kerak bo’lsa flip va fon rangi ni sozlang.
  5. Yuklab olish — bitta rasm uchun yoki barcha uchun ZIP.
Erkin burchak nima uchun kerak?

Telefon kamerasida qiyshiq tushgan rasmni to'g'irlash uchun. Masalan, gorizontni to'g'rilash uchun 2-3° aylantirish kerak bo'ladi.

Fon rangi nima uchun kerak?

90° dan farqli burchaklarda aylantirish paytida rasm to'rtburchak ichida joy qoladi. Shu bo'sh joy uchun rang tanlanadi: shaffof (PNG uchun), oq, qora yoki boshqa rang.

Sifat tushib qoladi?

90°, 180°, 270° aylantirishda piksel to'g'ridan to'g'ri ko'chiriladi — sifat deyarli o'zgarmaydi. Erkin burchakda bilinear interpolatsiya ishlatiladi — bu standart usul.

Flip nima?

Gorizontal flip — rasmni vertikal o'q bo'ylab ko'zgu ko'rinishiga keltiradi. Vertikal flip — gorizontal o'q bo'ylab. Ikkalasini birga qo'shish mumkin.

Rasm serverga yuboriladi?

Yo'q. Hamma jarayon JavaScript Canvas API orqali brauzeringizda bajariladi.
