About this tool

This tool opens Office documents from the inside and converts only the text content — design, tables, images, fonts, headers/footers and comments are never touched. The result differs from the original only in script.

Conversion follows the official 1995 Uzbek Latin alphabet rules: sh, ch, ng, oʻ, gʻ digraphs, the tutuq sign (maʼno → маъно), contextual е/э and ц rules, plus an exceptions dictionary (сентябрь → sentabr).

Files are never uploaded to a server — the whole process runs in your browser. Safe even for confidential and official documents.

Frequently asked questions

Will the formatting break?

No. The tool never touches document structure — only text characters are replaced. Tables, images, fonts, colors and layout are 100% preserved.

Is the direction detected automatically?

No — you choose the direction yourself, which prevents accidental wrong conversion. Default: Latin → Cyrillic.

Do legacy .doc / .xls / .ppt files work?

Not directly. Open them in Office and save in the modern format (.docx / .xlsx / .pptx), then upload.

Can scanned documents be converted?

If the text in the document is an image (a scan), it cannot be read — that requires OCR. The tool will tell you if this is the case.

Is there a file size limit?

Practically none — everything runs on your device, so documents up to 50 MB work fine.

What about plain text?

Use the Latin ⇄ Cyrillic text converter — it converts live as you type.
